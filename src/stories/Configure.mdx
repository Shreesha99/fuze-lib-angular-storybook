# Welcome to FuzeLib: Your Angular Component Library

FuzeLib is a versatile and robust Angular component library designed to accelerate your development process and enhance the user experience. With a rich set of prebuilt, reusable UI components, FuzeLib empowers developers to create consistent, scalable, and visually appealing applications effortlessly.

---

## **What is FuzeLib?**

FuzeLib is a modern Angular package created to address the common needs of web developers and designers. It provides essential UI components such as:

- **Button**: Customizable buttons with various styles and actions.
- **Accordion**: A collapsible container for hierarchical information.
- **DropdownList**: Feature-rich dropdown menus for data selection.
and more...

Each component is crafted with flexibility and performance in mind, ensuring seamless integration and responsiveness across devices.

## **Getting Started**

Follow these steps to integrate FuzeLib into your Angular project:

### **Installation**
Install FuzeLib using npm:
```bash
npm install fuze-lib-angular
```

### **Importing Components**

Import the desired component into your Angular module:

```bash
import { ButtonComponent } from 'fuze-lib-angular';

@NgModule({
  declarations: [/* Your Components */],
  imports: [ButtonComponent],
})
export class AppModule {}

```

### **Using Components**

Add the component to your template:

```bash
<lib-button 
[primary]="true"
[size]="'large'"
[backgroundColor]="'#DC3E4E'"
[borderRadius]="'0.5em'"
(onClick)="handleClick($event)">
</lib-button>

```
